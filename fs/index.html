สำคัญ: ต้องตั้งค่า GAS URL ใน index.html ก่อนใช้งานจริง
  1. ไปที่ script.google.com สร้าง Project ใหม่
  2. วางโค้ดจากไฟล์ Code.gs ลงใน Editor
  3. รันฟังก์ชัน setupSheets() เพื่อสร้าง Google Sheets
  4. Deploy > New Deployment > Web App
    • Execute as: Me
    • Who has access: Anyone
  5. คัดลอก Deployment URL
  6. แทนที่ค่า GAS_URL ใน index.html
  7. อัปโหลด index.html ขึ้น Web Host

วิธีเชื่อมต่อ Google Apps Script

ทำตามขั้นตอนเหล่านี้เพื่อเปิดใช้งานระบบเต็มรูปแบบ

1
สร้าง Google Sheet
เปิด Google Sheets สร้าง Spreadsheet ใหม่ แล้วคัดลอก ID จาก URL
https://docs.google.com/spreadsheets/d/[SPREADSHEET_ID]/edit
2
สร้าง Google Apps Script
เปิด script.google.com > New Project > วาง Code.gs > บันทึก
3
แก้ไข SPREADSHEET_ID
ใน Code.gs บรรทัดแรก ใส่ ID ของ Google Sheet ที่สร้างไว้
4
Deploy Web App
Deploy > New Deployment > Web App
• Execute as: Me
• Who has access: Anyone
5
อัปเดต CONFIG.GAS_URL
Copy URL ที่ได้จาก Deploy มาใส่ใน index.html บรรทัด CONFIG.GAS_URL
6
ตั้งพิกัดสำนักงาน
แก้ OFFICE_LAT, OFFICE_LNG ทั้งใน index.html และ Code.gs ให้ตรงกับที่ตั้งสำนักงาน
Host index.html บน GitHub Pages เพื่อให้กล้องทำงานได้ (HTTPS required)
